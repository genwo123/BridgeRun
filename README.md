### 🏗️ **무엇을 하는 게임인가?**
- **3-4팀 × 3명**이 동시에 경쟁
- **판자로 다리 건설** → 적 팀 방해
- **토템 쟁탈** → 점수 획득
- **15분 3라운드** 승부

### ⚡ **왜 재미있는가?**
- **협력 + 경쟁** 동시 진행
- **실시간 건설/파괴** 역학
- **5가지 직업** 역할 분담
- **순간의 판단**이 승부 결정
